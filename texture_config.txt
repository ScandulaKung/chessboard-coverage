# Piece Texture Configuration File
# =================================
# This file allows you to specify custom texture paths for each piece.
# 
# Format: piece_name=texture_file_path
# 
# If a piece is not specified here, the program will use the default path:
#   resources/piece_name.png
#
# Examples:
#   3x3=custom_textures/my_3x3.png
#   L-shape=resources/custom_L.png
#   T-shape=C:/MyTextures/T_shape_custom.png
#
# Notes:
# - Lines starting with # are comments and will be ignored
# - Empty lines are also ignored
# - Paths can be absolute or relative to the program directory
# - If the specified texture file doesn't exist, the program will fall back
#   to the default path (resources/piece_name.png)
# - If the default path also doesn't exist, the program will create a default
#   colored texture automatically
#
# Available piece names:
#   3x3, 3x3L, 2x4, 2x3, L-shape, L-mirror, Z-mirror, line4, cross, T-shape,
